--- title: FreeCol 0.7.2 Released author: Stian Grenborgen --- We have now released version 0.7.2 of FreeCol. This release fixes several bugs and adds some new features as well. Some of the major new features are:
  • The game can now switch between fullscreen and windowed mode.
  • The application window can be resized.
  • Zooming on the mapboard has been implemented.
